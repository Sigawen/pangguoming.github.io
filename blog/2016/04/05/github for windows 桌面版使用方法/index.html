github for windows 桌面版使用方法

发表于 2016-04-05

首先得去下载github for windows，搜索下GitHub Desktop就可以找到，然后安装

打开软件进入主界面，选择右上角的设置按钮，会弹出下拉菜单，选择options

没一个帐号绑定，点击添加帐号，

进入登录帐号界面，填上帐号及密码

登录成功后转入主界面，可以下载个已经有的项目，选择左上解的+号，再选择clone，选择个自己的项目就可以下载下来了，

要创建一个新的项目的话，在+号后要选择Create，并选择本地关联路径

如果是已经存在的项目想上传上去的话，在+号后选择Add，并选择存在的项目

当本地改动时选择那个项目会有改动记录，在下面的提交上填上日志就可以提交上去了。
