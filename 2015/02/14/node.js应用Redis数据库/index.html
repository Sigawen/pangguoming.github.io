node.js应用Redis数据库

发表于 2015-02-14

node.js下使用Redis，首先：

1、有一台安装了Redis的服务器，当然，安装在本机也行

2、本机，也就是客户端，要装node.js

3、项目要安装nodejs_redis模块

注意第 3 点，不是在本机安装就行了，而是说，要在项目中安装（引用）。

方法是，DOS窗口，在项目目录下，输入

npm install redis

这样就将nodejs_redis下载一份，放到当前目录下了。看看，多了一个文件夹：node_modules\redis
